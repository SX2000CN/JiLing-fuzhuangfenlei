# 字体实施检查清单

本文档提供 MiSans 字体在项目中实施的完整检查清单，确保字体正确部署和优化。

## 📋 部署前检查

### 1. 字体文件准备

- [ ] 从官方渠道下载 MiSans 字体
- [ ] 确认字体版本（记录版本号）
- [ ] 选择需要的字重（推荐 4-5 个）
- [ ] 将 TTF 转换为 WOFF2 格式
- [ ] 验证转换后的文件完整性
- [ ] 按规范命名文件

**字重选择参考：**
| 场景 | 推荐字重 |
|------|---------|
| 基础项目 | Regular (400), Medium (500), Bold (700) |
| 标准项目 | Regular, Medium, Semibold (600), Bold |
| 完整项目 | Light (300), Regular, Medium, Semibold, Bold, Heavy (800) |

### 2. 文件组织

- [ ] 创建 `fonts/` 目录
- [ ] 字体文件放置正确位置
- [ ] 文件权限设置正确（可读）
- [ ] 目录结构清晰

```
project/
├── fonts/
│   ├── MiSans-Regular.woff2
│   ├── MiSans-Medium.woff2
│   ├── MiSans-Semibold.woff2
│   └── MiSans-Bold.woff2
├── css/
│   └── fonts.css
└── index.html
```

### 3. CSS 配置

- [ ] @font-face 声明正确
- [ ] 字体路径正确
- [ ] font-weight 值匹配
- [ ] font-display 已设置
- [ ] 后备字体栈完整

```css
/* 检查模板 */
@font-face {
  font-family: "MiSans";                    /* ✓ 字体名称 */
  src: url("/fonts/MiSans-Regular.woff2")   /* ✓ 路径正确 */
       format("woff2");                     /* ✓ 格式声明 */
  font-weight: 400;                         /* ✓ 字重值 */
  font-style: normal;                       /* ✓ 样式 */
  font-display: swap;                       /* ✓ 加载策略 */
}
```

### 4. 性能优化

- [ ] 使用 WOFF2 格式（最小体积）
- [ ] 实施字体子集化（如适用）
- [ ] 添加字体预加载
- [ ] 配置浏览器缓存
- [ ] 仅加载必要字重

## 🧪 功能测试

### 5. 基础显示测试

- [ ] 中文字符正常显示
- [ ] 英文字符正常显示
- [ ] 数字和符号正常显示
- [ ] 所有字重正确渲染
- [ ] 无乱码或方块字

### 6. 跨平台测试

**桌面浏览器：**
- [ ] Chrome (Windows)
- [ ] Chrome (macOS)
- [ ] Firefox
- [ ] Safari (macOS)
- [ ] Edge

**移动设备：**
- [ ] iOS Safari
- [ ] Chrome (Android)
- [ ] 微信内置浏览器

### 7. 降级测试

- [ ] 禁用 JavaScript 后字体正常
- [ ] 网络慢速下后备字体生效
- [ ] 字体加载失败时页面可用
- [ ] 无严重的字体闪烁问题

## 🔍 质量检查

### 8. 视觉一致性

- [ ] 字体渲染平滑（无锯齿）
- [ ] 字重对比明显
- [ ] 行高和间距合适
- [ ] 中英文混排对齐
- [ ] 不同字号下清晰可读

### 9. 无障碍检查

- [ ] 正文字号 ≥ 16px
- [ ] 细体字重仅用于大字号
- [ ] 足够的颜色对比度
- [ ] 支持 200% 缩放
- [ ] 行高 ≥ 1.5

### 10. 性能指标

- [ ] 字体文件总大小 < 2MB（子集化后）
- [ ] 首字节时间 (TTFB) 合理
- [ ] 首次内容绘制 (FCP) 未明显延迟
- [ ] Lighthouse 性能评分 ≥ 90

## 📦 部署检查

### 11. 服务器配置

- [ ] MIME 类型正确设置
  ```
  .woff2 → font/woff2
  .woff  → font/woff
  .ttf   → font/ttf
  ```
- [ ] 启用 Gzip/Brotli 压缩
- [ ] 配置长期缓存（Cache-Control）
- [ ] CORS 头设置正确（如跨域）

### 12. CDN 配置（如使用）

- [ ] 字体文件已上传到 CDN
- [ ] CDN 路径正确
- [ ] 启用 HTTPS
- [ ] 边缘缓存配置

### 13. 版本管理

- [ ] 字体文件已纳入版本控制（或 LFS）
- [ ] 记录字体版本号
- [ ] 更新日志已记录
- [ ] 备份原始 TTF 文件
