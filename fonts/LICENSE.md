# MiSans 字体许可证说明

## 📜 许可证概述

MiSans 字体由小米公司发布，采用 **SIL Open Font License 1.1 (OFL-1.1)** 开源许可证。

### 许可证关键信息

| 项目 | 说明 |
|------|------|
| 许可证类型 | SIL Open Font License 1.1 |
| 版权所有者 | 小米科技有限责任公司 |
| 官方网站 | https://hyperos.mi.com/font |
| 商业使用 | ✅ 允许 |
| 修改字体 | ✅ 允许（需使用不同名称） |
| 再分发 | ✅ 允许（需附带许可证） |
| 单独销售字体 | ❌ 不允许 |

## ✅ 允许的使用方式

### 1. 商业项目
- 网站和 Web 应用
- 移动应用（iOS/Android）
- 桌面应用程序
- 印刷品（海报、名片、书籍等）
- 广告和营销材料
- 游戏和多媒体

### 2. 嵌入和分发
- 嵌入到应用程序中
- 作为 Web 字体在网站上使用
- 包含在开源项目中
- 在 CDN 上托管

### 3. 修改和衍生
- 创建字体子集
- 转换字体格式（TTF → WOFF2）
- 修改字体以适应特定需求
- 基于 MiSans 创建衍生字体（需更名）

## ❌ 禁止的使用方式

### 1. 单独销售
- 不能将 MiSans 字体文件单独作为商品销售
- 但可以将其作为产品的一部分（如软件、设计模板）销售

### 2. 使用保留名称
- 修改后的字体不能继续使用 "MiSans" 名称
- 衍生字体必须使用新名称

## 📋 合规使用清单

在项目中使用 MiSans 字体时，请确保：

- [ ] 保留字体文件中的版权声明
- [ ] 在项目文档中注明使用了 MiSans 字体
- [ ] 如果分发字体文件，需附带 OFL 许可证
- [ ] 修改后的字体已更名（如适用）
- [ ] 没有单独销售字体文件

## 📝 推荐的版权声明

在项目中使用 MiSans 时，建议在以下位置添加声明：

### 网站 / Web 应用

在页面底部、关于页面或设计系统文档中添加：

```
本站使用 MiSans 字体，由小米公司发布，采用 SIL Open Font License 1.1 许可证。
```

### 移动应用 / 桌面应用

在"关于"页面或开源许可列表中添加：

```
MiSans Font
Copyright (c) Xiaomi Inc.
Licensed under the SIL Open Font License, Version 1.1.
https://hyperos.mi.com/font
```

### 设计系统文档

