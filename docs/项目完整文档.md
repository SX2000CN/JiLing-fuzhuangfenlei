# 项目完整文档（现状与结构规划）

更新时间：2026-04-07

## 1. 项目定位

本项目是基于 PyTorch 的服装图片三分类桌面应用，类别为：主图、细节、吊牌。项目同时保留 CLI 与多套 GUI 实现。

## 2. 当前可运行入口

1. `launchers/run_classify.bat` -> `src/cli/classify_cli.py`
2. `run_new_gui.py` -> 默认 `src/gui/native_ui.py`
3. `run_new_gui.py --modular` -> `src/gui/app.py`
4. `run_new_gui.py --traditional` -> `src/gui/main_window.py`

## 3. 目录全景（按职责）

### 3.1 业务代码

```text
src/
├─ core/      # 模型工厂、训练器、分类器
├─ cli/       # 命令行入口
├─ gui/       # 三套 GUI 与组件/主题/页面
└─ utils/     # 配置与工具函数
```

### 3.2 运行资源

```text
config/   # YAML 配置
models/   # 模型文件与子目录
data/     # train/val/test
fonts/    # MiSans 字体
logs/     # 运行日志
outputs/  # 分类输出
```

### 3.3 工程与协作

```text
launchers/  # Windows 启动脚本
build/      # 打包脚本
tests/      # 测试
docs/       # 文档
.github/    # CI 与协作规范
openspec/   # 变更提案与规格
```
