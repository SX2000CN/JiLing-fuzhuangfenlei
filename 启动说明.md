# JiLing 服装分类系统 - 启动说明

## 🚀 快速启动

### 方法一：统一启动器（推荐）

```bash
# 启动现代化版本（推荐）
python launch.py modern

# 启动传统版本
python launch.py traditional

# 交互式选择
python launch.py
```

### 方法二：独立启动器

```bash
# 现代化版本
python launch_modern.py

# 传统版本  
python launch_traditional.py

# 直接启动（不关闭之前的实例）
python modern_gui_main.py
python gui_main.py
```

## 🔧 技术栈

### 现代化版本
- **界面框架**: PySide6
- **设计风格**: 类似 Ant Design
- **颜色方案**: 现代化蓝色主题
- **布局**: 左侧导航 + 顶部标题 + 卡片内容

### 传统版本
- **界面框架**: PySide6
- **设计风格**: 传统桌面应用
- **布局**: 标签页布局

### 后端核心
- **深度学习**: PyTorch + timm
- **模型架构**: EfficientNetV2, ConvNeXt, ResNet等
- **图像处理**: PIL, torchvision

## 🐛 故障排除

1. **模型加载失败**
   - 确保 `models/` 目录存在
   - 检查是否有 `.pth` 模型文件

2. **依赖缺失**
   ```bash
   pip install -r requirements.txt
   ```

3. **启动失败**
   - 检查Python环境
   - 确保PySide6已安装
   - 查看终端错误信息
