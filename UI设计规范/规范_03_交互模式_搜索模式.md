# 搜索模式

## 概述

搜索是用户快速定位内容的核心功能。良好的搜索设计能够帮助用户高效找到所需信息，提升整体使用体验。

## 设计原则

- **易于发现**：搜索入口明显可见
- **快速响应**：即时显示搜索结果
- **结果相关**：优先展示最相关内容
- **容错友好**：处理拼写错误和模糊查询

## 搜索入口

### 搜索框位置

| 平台 | 位置 | 样式 |
|------|------|------|
| 网页端 | 顶部导航栏 | 展开式/图标触发 |
| 移动端 | 顶部/下拉显示 | 全宽搜索框 |
| 桌面端 | 顶部/侧边栏 | 固定搜索框 |

### 搜索框样式

```
┌─────────────────────────────────────┐
│ 🔍  搜索关键词...              ✕    │
└─────────────────────────────────────┘
```

| 元素 | 规范 |
|------|------|
| 高度 | 40-48px |
| 图标 | 左侧搜索图标 20px |
| 清除 | 右侧清除按钮 |
| 圆角 | 8px 或全圆角 |
| 占位符 | 提示搜索范围 |

## 搜索交互

### 即时搜索

| 触发 | 延迟 | 场景 |
|------|------|------|
| 输入时 | 300ms 防抖 | 小数据量 |
| 回车时 | 立即 | 大数据量 |
| 按钮点击 | 立即 | 复杂搜索 |

### 搜索建议

```
┌─────────────────────────────────────┐
│ 🔍  iphone                     ✕    │
├─────────────────────────────────────┤
│ 🕐  iphone 15 pro              历史 │
│ 🕐  iphone 手机壳              历史 │
├─────────────────────────────────────┤
│ 🔥  iphone 15                  热门 │
│ 🔍  iphone 充电器              建议 │
│ 🔍  iphone 数据线              建议 │
└─────────────────────────────────────┘
```

### 建议类型

| 类型 | 图标 | 来源 |
|------|------|------|
| 历史记录 | 🕐 | 用户搜索历史 |
| 热门搜索 | 🔥 | 全站热门 |
| 智能建议 | 🔍 | 算法推荐 |
| 分类建议 | 📁 | 内容分类 |

## 搜索结果

### 结果布局

| 类型 | 适用场景 |
|------|----------|
| 列表 | 文章、商品 |
| 网格 | 图片、视频 |
| 混合 | 多类型内容 |

### 结果项结构

```
┌─────────────────────────────────────┐
│ [图片]  标题（高亮关键词）          │
│         描述文字...                 │
│         来源 · 时间                 │
└─────────────────────────────────────┘
```

### 关键词高亮

- 颜色：Primary-500 或加粗
- 范围：标题和描述
- 方式：精确匹配

### 结果排序

| 排序 | 描述 |
|------|------|
| 相关度 | 默认，最相关优先 |
| 时间 | 最新/最早 |
| 热度 | 浏览量/点赞数 |
| 价格 | 低到高/高到低 |

## 筛选与过滤

### 筛选器位置

| 平台 | 位置 |
|------|------|
| 网页端 | 左侧边栏/顶部 |
| 移动端 | 顶部筛选栏/底部面板 |
| 桌面端 | 左侧边栏 |

### 筛选器类型

| 类型 | 组件 | 场景 |
|------|------|------|
| 单选 | 单选按钮/下拉 | 排序、分类 |
| 多选 | 复选框 | 标签、属性 |
| 范围 | 滑块/输入框 | 价格、日期 |
| 搜索 | 输入框 | 品牌、作者 |

### 已选筛选

- 显示已选条件标签
- 支持单个移除
- 提供清除全部

## 空状态与错误

### 无结果

```
┌─────────────────────────────────────┐
│                                     │
│            [搜索图标]               │
│                                     │
│      未找到 "xxx" 相关结果          │
│                                     │
│      建议：                         │
│      · 检查关键词拼写               │
│      · 尝试更简短的关键词           │
│      · 尝试其他相关词               │
│                                     │
│      热门搜索：标签1 标签2 标签3    │
└─────────────────────────────────────┘
```

### 搜索错误

- 显示错误原因
- 提供重试按钮
- 保留搜索词

## 高级搜索

### 功能

- 多字段搜索
- 布尔运算（AND/OR/NOT）
- 精确匹配（引号）
- 范围限定

### 入口

- 搜索框旁"高级搜索"链接
- 筛选器展开
- 搜索语法提示

## 平台差异

### 网页端
- 支持快捷键（Ctrl/Cmd + K）
- 搜索结果分页
- 复杂筛选器
- URL 保存搜索状态

### 移动端
- 全屏搜索体验
- 语音搜索
- 扫码搜索
- 无限滚动加载

### 桌面端
- 全局搜索（Spotlight 式）
- 快捷键触发
- 多窗口结果

## 无障碍考虑

- 搜索框 `role="search"`
- 结果区域 `aria-live`
- 键盘可导航结果
- 屏幕阅读器播报结果数

## 最佳实践

### ✅ 推荐
- 保存搜索历史
- 提供搜索建议
- 高亮匹配关键词
- 支持拼写纠错

### ❌ 避免
- 隐藏搜索入口
- 无结果无引导
- 搜索响应过慢
- 筛选器过于复杂
