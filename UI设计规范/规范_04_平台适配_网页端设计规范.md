# 网页端设计规范

## 概述

网页端设计需要考虑多种浏览器、屏幕尺寸和输入方式。本规范定义了桌面浏览器和响应式网页的设计标准。

## 设计原则

- **响应式优先**：适配各种屏幕尺寸
- **渐进增强**：基础功能优先保障
- **性能意识**：优化加载和交互性能
- **可访问性**：支持各类用户和设备

## 布局规范

### 页面结构

```
┌────────────────────────────────────────────────┐
│                    Header (64px)               │
├──────────┬─────────────────────────────────────┤
│          │                                     │
│ Sidebar  │           Main Content              │
│ (240px)  │                                     │
│          │                                     │
├──────────┴─────────────────────────────────────┤
│                    Footer                      │
└────────────────────────────────────────────────┘
```

### 内容宽度

| 类型 | 最大宽度 | 用途 |
|------|----------|------|
| 窄内容 | 680px | 文章阅读 |
| 标准内容 | 1200px | 通用页面 |
| 宽内容 | 1400px | 数据密集 |
| 全宽 | 100% | 仪表盘 |

### 响应式断点

| 断点 | 范围 | 布局调整 |
|------|------|----------|
| xs | < 576px | 单列，隐藏侧边栏 |
| sm | 576-767px | 单列，简化导航 |
| md | 768-991px | 可选侧边栏 |
| lg | 992-1199px | 完整布局 |
| xl | ≥ 1200px | 宽松布局 |

## 导航设计

### 顶部导航

| 元素 | 规范 |
|------|------|
| 高度 | 64px |
| Logo | 左侧，高度 32-40px |
| 导航项 | 居中或右侧 |
| 操作区 | 右侧（搜索、用户） |

### 侧边导航

| 状态 | 宽度 | 触发 |
|------|------|------|
| 展开 | 240px | 默认 |
| 收起 | 64px | 点击切换 |
| 隐藏 | 0 | 小屏幕 |

### 面包屑

- 位置：内容区顶部
- 分隔符：`/` 或 `>`
- 最后一项不可点击

## 交互规范

### 鼠标交互

| 元素 | 悬停效果 |
|------|----------|
| 链接 | 颜色变化+下划线 |
| 按钮 | 背景色加深 |
| 卡片 | 阴影增强 |
| 表格行 | 背景色变化 |

### 键盘支持

| 功能 | 要求 |
|------|------|
| Tab 导航 | 所有交互元素可达 |
| 焦点指示 | 清晰可见的焦点环 |
| 快捷键 | 常用操作支持 |
| 表单提交 | Enter 键提交 |

### 滚动行为

| 类型 | 实现 |
|------|------|
| 平滑滚动 | scroll-behavior: smooth |
| 锚点定位 | 预留顶部导航高度 |
| 返回顶部 | 滚动后显示按钮 |
| 无限滚动 | 可选，提供分页替代 |

## 表单设计

### 布局

| 屏幕 | 布局 |
|------|------|
| 大屏 | 双列，标签左侧 |
| 中屏 | 单列，标签顶部 |
| 小屏 | 单列，标签顶部 |

### 验证

- 实时验证（失焦时）
- 提交前全量验证
- 错误信息内联显示
- 成功状态反馈

## 数据展示

### 表格

| 屏幕 | 处理方式 |
|------|----------|
| 大屏 | 完整表格 |
| 中屏 | 隐藏次要列 |
| 小屏 | 卡片列表或横向滚动 |

### 分页

- 显示总数和当前页
- 支持跳转指定页
- 可选每页条数

## 性能优化

### 加载策略

| 策略 | 应用 |
|------|------|
| 懒加载 | 图片、非首屏内容 |
| 预加载 | 关键资源 |
| 代码分割 | 按路由分割 |
| 缓存 | 静态资源 |

### 感知性能

- 骨架屏占位
- 渐进式图片加载
- 乐观更新
- 加载进度指示

## 浏览器兼容

### 支持范围

| 浏览器 | 最低版本 |
|--------|----------|
| Chrome | 最近2个版本 |
| Firefox | 最近2个版本 |
| Safari | 最近2个版本 |
| Edge | 最近2个版本 |

### 降级策略

- 功能检测优于浏览器检测
- 提供降级方案
- 关键功能保障

## 无障碍要求

| 要求 | 标准 |
|------|------|
| 对比度 | WCAG AA (4.5:1) |
| 键盘 | 完全可操作 |
| 屏幕阅读器 | 语义化标签 |
| 缩放 | 支持 200% |

## 最佳实践

### ✅ 推荐
- 移动优先设计
- 语义化 HTML
- 渐进增强
- 性能监控

### ❌ 避免
- 固定宽度布局
- 仅鼠标可用
- 忽视加载状态
- 过度依赖 JavaScript
