# 手势与动作

## 概述

手势与动作定义了用户与界面的物理交互方式。不同平台有不同的输入方式，设计时需要考虑触控、鼠标、键盘等多种交互模式。

## 设计原则

- **自然直觉**：手势符合物理世界直觉
- **可发现**：用户能发现手势功能
- **可逆转**：误操作可以撤销
- **一致性**：相同手势相同效果

## 触控手势

### 基础手势

| 手势 | 动作 | 常见用途 |
|------|------|----------|
| 点击（Tap） | 单指轻触 | 选择、激活 |
| 双击（Double Tap） | 快速点击两次 | 放大、点赞 |
| 长按（Long Press） | 按住不放 | 上下文菜单、拖拽 |
| 滑动（Swipe） | 单指快速划过 | 翻页、删除 |
| 拖拽（Drag） | 按住并移动 | 移动、排序 |
| 捏合（Pinch） | 双指捏合/张开 | 缩放 |
| 旋转（Rotate） | 双指旋转 | 旋转图片 |

### 手势参数

| 参数 | 值 |
|------|-----|
| 点击时长 | < 300ms |
| 长按时长 | ≥ 500ms |
| 滑动距离 | ≥ 10px |
| 滑动速度 | ≥ 0.3px/ms |

### 边缘手势

| 手势 | 位置 | 用途 |
|------|------|------|
| 左边缘滑动 | 屏幕左侧 | 返回（iOS） |
| 右边缘滑动 | 屏幕右侧 | 前进 |
| 顶部下拉 | 屏幕顶部 | 刷新、通知 |
| 底部上滑 | 屏幕底部 | 主屏幕（iOS） |

## 键盘交互

### 导航键

| 按键 | 功能 |
|------|------|
| Tab | 下一个焦点 |
| Shift + Tab | 上一个焦点 |
| Enter | 激活/提交 |
| Space | 激活/选择 |
| Escape | 取消/关闭 |
| 方向键 | 列表/菜单导航 |

### 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl/Cmd + S | 保存 |
| Ctrl/Cmd + Z | 撤销 |
| Ctrl/Cmd + Y | 重做 |
| Ctrl/Cmd + C | 复制 |
| Ctrl/Cmd + V | 粘贴 |
| Ctrl/Cmd + A | 全选 |
| Ctrl/Cmd + F | 搜索 |
| Ctrl/Cmd + K | 命令面板 |

### 焦点管理

| 规则 | 说明 |
|------|------|
| 可见焦点 | 焦点环清晰可见 |
| 逻辑顺序 | 从左到右、从上到下 |

| 焦点恢复 | 关闭后返回触发元素 |

## 拖拽交互

### 拖拽类型

| 类型 | 用途 |
|------|------|
| 排序拖拽 | 列表项重排 |
| 移动拖拽 | 元素位置移动 |
| 文件拖拽 | 文件上传 |
| 跨区域拖拽 | 分类移动 |

### 拖拽反馈

| 阶段 | 反馈 |
|------|------|
| 开始 | 元素半透明/阴影 |
| 拖拽中 | 跟随光标/手指 |
| 悬停目标 | 目标区域高亮 |

### 拖拽规范

| 元素 | 规范 |
|------|------|
| 拖拽手柄 | 6点图标，左侧 |
| 拖拽预览 | 原元素缩小版 |
| 放置指示 | 线条或高亮区域 |

## 滚动行为

### 滚动类型

| 类型 | 描述 |
|------|------|
| 自然滚动 | 内容跟随手指 |
| 惯性滚动 | 松手后继续滚动 |
| 弹性滚动 | 边界回弹效果 |
| 吸附滚动 | 滚动到固定位置 |

### 滚动优化

- 平滑滚动（smooth scroll）
- 滚动锚定
- 虚拟滚动（长列表）
- 滚动节流

## 平台差异

### 网页端
- 鼠标+键盘为主
- 支持悬停状态
- 右键菜单
- 滚轮滚动

### 移动端
- 触控为主
- 无悬停状态
- 长按替代右键
- 系统手势优先

### 桌面端
- 鼠标+键盘+触控板
- 丰富快捷键
- 多点触控手势
- 精确操作

## 无障碍考虑

- 所有功能键盘可达
- 提供手势替代方案
- 焦点指示清晰
- 支持辅助技术

## 最佳实践

### ✅ 推荐
- 使用标准手势
- 提供视觉反馈
- 支持键盘操作
- 手势可发现

### ❌ 避免
- 自定义冲突手势
- 无反馈的操作
- 仅支持触控
- 隐藏的手势功能
