# 表单元素

## 概述

表单元素是用户输入数据的核心组件。良好的表单设计能够提高数据采集效率，减少用户错误，提升整体体验。

## 设计原则

- **简洁高效**：仅收集必要信息
- **清晰引导**：标签和提示明确
- **及时反馈**：验证结果即时显示
- **容错性强**：支持多种输入格式

## 输入框（Input）

### 类型

| 类型 | 用途 | HTML type |
|------|------|-----------|
| 文本 | 通用文字输入 | text |
| 密码 | 密码输入 | password |
| 邮箱 | 邮箱地址 | email |
| 电话 | 电话号码 | tel |
| 数字 | 数值输入 | number |
| 搜索 | 搜索查询 | search |
| 网址 | URL 输入 | url |

### 尺寸规范

| 尺寸 | 高度 | 内边距 | 字号 |
|------|------|--------|------|
| sm | 32px | 8px 12px | 14px |
| md | 40px | 10px 16px | 14px |
| lg | 48px | 12px 16px | 16px |

### 状态

| 状态 | 边框色 | 背景色 | 说明 |
|------|--------|--------|------|
| Default | Gray-300 | White | 默认 |
| Hover | Gray-400 | White | 悬停 |
| Focus | Primary-500 | White | 聚焦，2px边框 |
| Disabled | Gray-200 | Gray-50 | 禁用 |
| Error | Error-500 | Error-50 | 验证失败 |
| Success | Success-500 | White | 验证通过 |

## 选择器（Select）

### 下拉选择

| 元素 | 规范 |
|------|------|
| 高度 | 同输入框 |
| 下拉图标 | 16-20px，右侧 |
| 下拉面板 | shadow-lg，最大高度 300px |
| 选项高度 | 36-40px |

### 多选选择器

- 选中项显示为标签
- 支持搜索过滤
- 可设置最大选择数

| 尺寸 | 框大小 | 图标 |
|------|--------|------|
| sm | 16×16px | 12px |
| md | 20×20px | 14px |
| lg | 24×24px | 18px |

### 状态

| 状态 | 描述 |
|------|------|
| Unchecked | 空心边框 |
| Checked | 填充主色 + 勾选图标 |

## 单选框（Radio）

### 规范

- 尺寸同复选框
- 选中显示实心圆点
- 圆点大小为框的 50%
- 同组单选项间距 12-16px

## 表单布局

### 标签位置

| 位置 | 适用场景 |
|------|----------|
| 顶部 | 移动端、长标签 |
| 左侧 | 桌面端、简短标签 |
| 内联 | 紧凑空间、Material风格 |

### 字段间距

| 元素 | 间距 |
|------|------|
| 标签与输入框 | 4-8px |
| 输入框与提示 | 4px |
| 字段之间 | 16-24px |
| 分组之间 | 32px |

## 表单验证

### 验证时机

| 时机 | 场景 |
|------|------|
| onChange | 实时验证，简单规则 |
| onBlur | 失焦验证，复杂规则 |
| onSubmit | 提交前全量验证 |

### 错误提示

- 位置：输入框下方
- 颜色：Error-500
- 图标：可选错误图标
- 文案：说明问题和解决方法

## 平台差异

### 网页端
- 支持自动填充（autocomplete）
- 可使用自定义样式
- 支持键盘导航

### 移动端
- 使用合适的键盘类型
- 标签置于顶部
- 考虑使用原生选择器
- 输入时自动滚动到可见区域

### 桌面端
- 可使用更紧凑布局
- 支持 Tab 快速切换
- 下拉可使用键盘搜索

## 无障碍考虑

- 每个输入关联 `<label>`
- 使用 `aria-describedby` 关联提示
- 错误状态使用 `aria-invalid`
- 必填字段使用 `required` 和 `aria-required`
- 表单分组使用 `<fieldset>` 和 `<legend>`

## 最佳实践

### ✅ 推荐
- 标签简短明确
- 提供输入示例或格式提示
- 实时验证并即时反馈
- 自动聚焦第一个字段

### ❌ 避免
- 使用 placeholder 替代标签
- 过长的表单不分组
- 只在提交时才显示错误
- 必填字段不标注
