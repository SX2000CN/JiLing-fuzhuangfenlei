# 容器与卡片

## 概述

容器与卡片是用于组织和分组内容的布局组件。它们帮助建立视觉层次，区分不同内容区域，提升界面的可读性和美观度。

## 设计原则

- **内容优先**：容器服务于内容，不喧宾夺主
- **层次分明**：通过视觉差异区分内容层级
- **一致性**：相同类型内容使用相同容器样式
- **适度装饰**：避免过度使用边框和阴影

## 卡片（Card）

### 基础结构

```
┌─────────────────────────────────────┐
│  [图片区域]                          │  ← 可选
├─────────────────────────────────────┤
│  标题                                │
│  副标题或描述文字                     │
│                                     │
│  内容区域...                         │
├─────────────────────────────────────┤
│  [操作区域]              [ 按钮 ]    │  ← 可选
└─────────────────────────────────────┘
```

### 尺寸规范

| 元素 | 规范 |
|------|------|
| 内边距 | 16px - 24px |
| 圆角 | 8px - 12px |
| 阴影 | shadow（默认） |
| 边框 | 可选，1px Gray-200 |

### 卡片类型

| 类型 | 特征 | 适用场景 |
|------|------|----------|
| 基础卡片 | 白色背景+阴影 | 通用内容 |
| 边框卡片 | 边框无阴影 | 列表项 |
| 可点击卡片 | 悬停效果 | 导航入口 |
| 选择卡片 | 选中状态 | 选项选择 |
| 媒体卡片 | 顶部图片 | 商品、文章 |

### 状态

| 状态 | 样式变化 |
|------|----------|
| 默认 | shadow |
| 悬停 | shadow-md，可选上移 |
| 选中 | Primary 边框 |
| 禁用 | 灰色背景，降低透明度 |

### 卡片间距

| 布局 | 间距 |
|------|------|
| 网格布局 | 16px - 24px |
| 列表布局 | 12px - 16px |
| 紧凑布局 | 8px |

## 面板（Panel）

### 用途

页面中的独立内容区块。

### 结构

```
┌─────────────────────────────────────┐
│  面板标题                    [ 操作 ]│
├─────────────────────────────────────┤
│                                     │
│              内容区域                │
│                                     │
└─────────────────────────────────────┘
```

### 规范

| 元素 | 规范 |
|------|------|
| 头部高度 | 48-56px |
| 头部边框 | 底部 1px Gray-200 |
| 内边距 | 16px - 24px |
| 背景 | White |
| 圆角 | 8px |

## 折叠面板（Collapse/Accordion）

### 规范

| 元素 | 规范 |
|------|------|
| 头部高度 | 48-56px |
| 展开图标 | 16-20px，右侧 |
| 内容内边距 | 16px |
| 边框 | 1px Gray-200 |

### 行为

- 单个展开 / 多个展开
- 动画时长：200-300ms
- 图标旋转指示状态

## 分割线（Divider）

### 类型

| 类型 | 用途 |
|------|------|
| 水平分割 | 区分上下内容 |
| 垂直分割 | 区分左右内容 |
| 带文字分割 | 分组标题 |

### 规范

| 元素 | 规范 |
|------|------|
| 颜色 | Gray-200 |
| 粗细 | 1px |
| 间距 | 上下 16-24px |

## 容器（Container）

### 页面容器

| 类型 | 最大宽度 | 用途 |
|------|----------|------|
| sm | 640px | 文章阅读 |
| md | 768px | 表单页面 |
| lg | 1024px | 标准页面 |
| xl | 1280px | 宽屏页面 |
| full | 100% | 全宽布局 |

### 内容容器

| 元素 | 规范 |
|------|------|
| 水平内边距 | 16px（移动）/ 24px（桌面） |
| 居中 | margin: 0 auto |

## 区块（Section）

### 用途

页面中的大型内容分区。

### 规范

| 元素 | 规范 |
|------|------|
| 垂直间距 | 48px - 64px |
| 背景 | 可交替使用 Gray-50 |
| 标题 | 居中或左对齐 |

## 平台差异

### 网页端
- 使用阴影区分层级
- 支持悬停效果
- 可使用更大内边距

### 移动端
- 减少阴影使用
- 使用边框或背景色区分
- 卡片可全宽显示
- 内边距适当减小

### 桌面端
- 可使用更丰富的层级
- 支持拖拽排序卡片
- 面板可折叠/展开

## 无障碍考虑

- 可点击卡片使用 `role="button"`
- 折叠面板使用 `aria-expanded`
- 分组内容使用 `role="group"`
- 确保足够的颜色对比度

## 最佳实践

### ✅ 推荐
- 相关内容放在同一卡片
- 保持卡片内容简洁
- 使用一致的卡片样式
- 合理使用留白

### ❌ 避免
- 卡片嵌套过深
- 过多装饰元素
- 内容过于拥挤
- 不一致的圆角和阴影
