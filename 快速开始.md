# 🚀 快速使用指南

## 最简单的启动方式

### 双击 `start.bat` 

这是推荐的启动方式!会显示一个菜单:

```
========================================
  JiLing 服装分类系统
========================================

请选择启动方式:

[1] 启动现代化界面 (推荐)
[2] 启动传统界面
[3] 命令行分类工具
[4] 测试环境
[5] 退出

========================================
```

输入数字 1-5 选择功能即可!

---

## 其他启动方式

### 快捷启动 (推荐)
- **现代界面**: 双击 `启动-现代界面.bat`
- **传统界面**: 双击 `启动-传统界面.bat`

### 从 launchers 目录启动
进入 `launchers/` 目录,有更多批处理选项:
- `快速启动-现代界面.bat` - 现代化 UI
- `快速启动-传统界面.bat` - 传统 UI
- `run_classify.bat` - 命令行分类
- `测试环境.bat` - 检查环境

### Python 命令启动
```bash
# 现代界面
python launchers/launch.py --mode modern

# 传统界面  
python launchers/launch.py --mode traditional

# 命令行分类
python classify_cli.py
```

## 🔧 常见问题

### Q: 启动失败怎么办?
A: 先运行 `launchers\测试环境.bat` 检查环境

### Q: 找不到模型文件?
A: 确保 `models/` 目录下有 `.pth` 模型文件

### Q: 想切换界面风格?
A: 使用主菜单 `start.bat` 选择不同的启动方式

### Q: 如何批量处理图片?
A: 选择菜单选项 [3] 命令行分类工具
